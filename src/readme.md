# 통계적 분석 Python 코드 리포지토리

이 리포지토리는 통계적 분석과 관련된 다양한 내용을 정리한 Python 코드들을 포함하고 있습니다. 통계학과 데이터 분석에 관심 있는 분들에게 학습 자료로 활용될 수 있도록 구성하였습니다.

## 주요 내용

- **기초 통계**  
  - 평균, 중앙값, 최빈값 계산
  - 표준편차 및 분산 계산
  - 데이터 분포 확인

- **가설 검정**  
  - t-검정, 카이제곱 검정, ANOVA
  - p-value 계산
  - 신뢰 구간 추정

- **회귀 분석**  
  - 단순 선형 회귀

- **시각화**  
  - 히스토그램, 박스플롯, 산점도
  - 상관계수 히트맵
  - 추세선 그리기
